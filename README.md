Electricity Bill Generator (C)
A basic C program that simulates monthly electricity billing for residential users based on UPCL (Uttarakhand) tariff slabs. It calculates the total bill using energy slabs, fixed charges, surcharges, and optional late fees.

Features
1. Add multiple customers (ID, name, units, load)
2. Slab-based bill calculation
3. Includes duty & surcharge
4. Optional late payment fee
5. Prints a simple receipt

Current Status
