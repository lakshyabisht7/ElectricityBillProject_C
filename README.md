Electricity Bill Generator
A mini-project in C programming that simulates monthly electricity billing for residential users based on UPCL (Uttarakhand) tariff slabs. It calculates the total bill using energy slabs, fixed charges, surcharges, and optional late fees. It utilizes concepts like structures, file handling, pointers, functions and searching.

Features
1. Add multiple customers (ID, name, units, load) 
2. Slab-based bill calculation
3. Includes duty & surcharge
4. Optional late payment fee
5. Prints a formatted bill receipt
6. Saves customers' information, using file handling
7. Searches customer records by ID or name
